How to use this layout planner

  1. Add your stories — paste titles one per line in the Manual entry tab and click Add stories. Or switch to Import CSV and drag in your Google Sheets export. The app reads the Article, Section, and Pg Ct columns automatically.
  2. Set types and page counts — each story gets two dropdowns: one for type (Feature, Boilerplate, etc.) and one for how many pages it spans. CSV imports make a best guess at type from the Section column — adjust any that look off.
  3. Build the layout — click Build layout to flow stories into the spread grid in list order. Multi-page stories show a [Np] badge and a faint "cont." on continuation pages.
  4. Adjust and iterate — change any type or page count, then click Build layout again. Set Total pages to match your issue.
  5. Export — Export PDF opens your browser's print dialog; choose "Save as PDF." Export text downloads a plain-text runsheet.
Color code: blue = Feature, green = Boilerplate, orange = Ad, red = Front/Cover story, purple = IFC/IBC, gray = Back cover. The counter in the toolbar shows pages used vs. remaining.
