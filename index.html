How to use this layout planner

  1. Add stories — paste titles one per line in Manual entry and click Add stories. Or switch to Import CSV and drag in your Google Sheets export. The app reads the Article, Section, and Pg Ct columns automatically.
  2. Set type & pages — each story gets two dropdowns: one for type (Feature, Boilerplate, etc.) and one for how many pages it spans.
  3. Build layout — click Build layout to flow stories into facing spreads in list order. Multi-page stories show all their spreads grouped together.
  4. Drag to reorder — grab the ⠿ handle on any story group and drag it up or down. All spreads for that story move together as one unit. Page numbers update automatically.
  5. Export — Export PDF opens your browser's print dialog; choose "Save as PDF." Export text downloads a plain-text runsheet.
Color code: blue = Feature · green = Boilerplate · orange = Ad · red = Front/Cover story · purple = IFC/IBC · gray = Back cover. The counter in the toolbar shows pages used vs. remaining.
